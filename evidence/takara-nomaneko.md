# タカラ事件・のまネコ問題 時系列記録

擬古猫・モナー等、2ちゃんねる系アスキーアートキャラクターの先使用・周知性に関わる2つの事件（2002年タカラ事件、2005年のまネコ問題）の時系列を、日本語版Wikipediaの記述に基づき整理したもの。Wikipedia自体が各事実に対しITmedia等の一次報道を出典として付しており、詳細確認が必要な場合はWikipedia側の脚注を辿ることができる。

- 出典1：[ギコ猫 - Wikipedia](https://ja.wikipedia.org/wiki/ギコ猫)（「ギコ猫商標問題（ギコ事件）」節）
- 出典2：[のまネコ問題 - Wikipedia](https://ja.wikipedia.org/wiki/のまネコ問題)
- 出典3：[strangewalker-gikoneko-2002.md](./strangewalker-gikoneko-2002.md)（当事者の同時代書き込み、特許電子図書館の出願情報転載を含む）

## タカラ事件（2002年）

出典1より。

- 2002-03-12：玩具メーカー・タカラ（現タカラトミー）が、ギコ猫のアスキーアートを対象として商標登録出願（商願2002-19166）。特許電子図書館の出願情報（出典3経由で確認）では、出願日・先願権発生日ともに2002-03-12、標準文字商標「ギコ猫」、称呼「ギコネコ」「ギコ」。
- （日付記載なし）：2ちゃんねる元管理人・西村博之がタカラへ抗議文を送付。
- 2002-06-03：タカラ、商標登録出願を取り下げ。
- この一件を機に、キャラクター「タカラギコ」が誕生（時期の明記なし）。

## のまネコ問題（2005年）

出典2より。

- 2005-03：O-Zoneのアルバムに収録曲「DiscO-Zone」（邦題「恋のマイアヒ」）収録。
- 2005-08-22：「恋のマイアヒ」がオリコン・アルバムチャート1位を記録。
- 2005-09-01：エイベックスの子会社エイベックスネットワークが、話題となっていたFlash動画のキャラクターを基に商用キャラクター「のまネコ」を作成、「(c)のまネコ製作委員会」の著作権表示を使用。アスキーアート「モナー」に似ているとの指摘が相次ぐ。
- 2005-09-08：エイベックスネットワークが自社サイトで見解を発表。「のまネコ」の著作権は有限会社ZENが管理し商品化契約を締結していること、モナー等の既存アスキーアートに着想を得て新たなオリジナリティを加えたものであること、既存AAキャラクターの使用制限の意図はないことを説明。
- 2005-09-22：特許電子図書館にて、有限会社ZENの商標登録申請が公開。
- 2005-09-24：西村博之が「のまタコ」グッズ展開の予定を告知し、問題がないかを確認する公開質問状をエイベックス・グループ・ホールディングスへ提出。
- 2005-09-30：エイベックス、「のまネコ」の商標登録中止とCD付録Flash動画の取りやめを発表。同日、同社社員への殺害予告について被害届提出の意向を表明。
- 2005-10-01：松浦勝人氏宅への放火をほのめかす書き込みが2ちゃんねる内に確認される。
- 2005-10-03：松浦氏の妻への濃硫酸使用をほのめかす書き込みが確認され、赤坂署が脅迫容疑で捜査を開始。同日、有限会社ZENがモナーの図案「米酒」の商標を取り下げ、エイベックスも該当Flash動画収録CDの回収を通達。
- 2005-10-04：2ちゃんねる管理人・西村博之がコメントを発表。
- 2005-10-07：O-Zoneがテレビ朝日「ミュージックステーション」に出演し「恋のマイアヒ」を熱唱。空耳日本語詞のテロップ使用について抗議が寄せられる（Wikipedia上「要出典」表記あり）。
- 2005-10-12：エイベックス、「のまネコ」グッズに関するキャラクター使用料の受け取りを辞退すると発表。
- 2005-11-01：仙台市内での児童殺害予告で逮捕されていた男が、エイベックス社員への殺害予告等の書き込みも行ったと供述。同22日、強要未遂容疑で再逮捕。
- 2005-12-05：同容疑者、松浦氏宅への放火容疑で再逮捕。
- 2006-01-20：同容疑者、名誉毀損容疑（親告罪）で追送検。
- 2006-02-03：東京地裁にて初公判。
- 2006-03-02：東京地裁にて第2回公判。懲役1年6ヶ月の実刑判決が言い渡され確定。
